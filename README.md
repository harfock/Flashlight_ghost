# Flashlight_ghost
This is a vibe coding game, built with Gemini Fast module. Use flashlight and laser to catch ghost. 
# Name of game: 🔦 Midnight Hunt: Shadow Combat (v1.0.4)

A modular, cross-platform "Vibe Coding" ghost hunter. Navigate a cluttered, haunted room using motion sensors, manage your failing hardware, and capture spirits to build your library.

## 🕹 Features
- **Gyro-Motion Controls**: Tilt your phone to scan the darkness.
- **Dynamic Sonar**: The "Bard" helper shifts music pitch as you approach ghosts.
- **Resource Management**: Balancing Laser Power vs. Flashlight Battery.
- **Emergency Crank**: A tactile mini-game to recharge when power fails.
- **Ghost Library**: Equip captured ghosts to gain unique stat boosts.
- **Cloud Saving**: Progress persists via PocketBase integration.

## 🛠 Tech Stack
- **Frontend**: Vanilla JS (ES6 Modules), HTML5 Canvas.
- **Backend**: PocketBase (hosted on PocketHost.io).
- **Deployment**: GitHub Pages (Frontend).

## 🚀 Deployment
1. Set up a free PocketBase instance at PocketHost.io.
2. Update `cloud.js` with your instance URL.
3. Push to GitHub and enable GitHub Pages.
